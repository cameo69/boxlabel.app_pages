## GitHub Pages

This branch is set up to be used with [GitHub Pages](https://pages.github.com/).

The `index.html` at the root of this branch will be served as the site homepage.

### Enabling GitHub Pages

1. Go to the repository **Settings** on GitHub.
2. Navigate to **Pages** in the left sidebar.
3. Under **Source**, select the branch containing this `index.html`.
4. Click **Save**.

Your site will be published at `https://<username>.github.io/test_bx_a/`.
