Hosting

Diese Website wird gehostet von GitHub Pages (GitHub, Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA). Beim Besuch der Seite protokollieren die Server von GitHub automatisch Standard-Zugriffsdaten wie IP-Adresse, Browsertyp und aufgerufene URL. Diese Daten werden von GitHub zu technischen Sicherheitszwecken gemäß deren Datenschutzerklärung verarbeitet. Wir haben keinen Zugriff auf diese Serverlogs.

Keine Cookies & kein Tracking

Diese Website setzt keine Cookies und verwendet keine Analysesoftware, Werbenetzwerke oder Tracking-Pixel. Die Spracheinstellung wird einmalig über die browsereigene navigator.language-API erkannt — dieser Wert wird weder gespeichert noch übertragen.

Datenspeicherung in der App

Die BoxLabel-App speichert alle von Ihnen erstellten Daten — Kistennamen, Inhalte und Fotos — ausschließlich auf Ihrem Gerät. Es werden keine Daten an externe Server übertragen. Ihre Daten verlassen Ihr Gerät nur dann, wenn Sie sie selbst exportieren oder teilen.

Kamerazugriff

Die App fordert Kamerazugriff an, um QR-Code-Etiketten zu scannen und Fotos von Kisteninhalten hinzuzufügen. Sämtliche Scan- und Aufnahmevorgänge finden ausschließlich auf dem Gerät statt. Es werden zu keinem Zeitpunkt Bilder oder Videoframes hochgeladen oder extern verarbeitet.

Universal Links

QR-Codes für Kisten kodieren eine URL der Form /box/UUID. Das Aufrufen eines solchen Links löst eine clientseitige JavaScript-Weiterleitung aus, die die entsprechende Kiste in der BoxLabel-App öffnet. Beim Aufruf dieses Links werden serverseitig keine personenbezogenen Daten erfasst. Die UUID dient ausschließlich als Bezeichner im Deep Link.
